# 水果连连看

一款运行在浏览器端的经典连连看游戏。无需安装、无需注册，打开即玩。

## 简介

严格还原经典两折角连线规则，支持提示、重排、计时等核心功能。纯前端实现，无服务器依赖，无广告干扰。

## 安装 / 运行

无需构建工具，直接在浏览器中打开 HTML 文件即可：

```bash
# 方式一：直接双击文件
open 水果连连看.html

# 方式二：用本地 HTTP 服务器（推荐，避免部分浏览器的 file:// 限制）
npx serve .
# 然后访问 http://localhost:3000
```

## 玩法说明

1. 点击一个图案选中它（蓝色高亮）
2. 再点击另一个相同图案，若两者之间的连线折角不超过两次，则消除成功
3. 全部消除完成前，用好「提示」和「重排」功能
4. 计时结束前完成消除即为胜利
