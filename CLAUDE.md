# CLAUDE.md — 水果连连看

## 项目定位

浏览器端经典连连看游戏，纯前端单文件实现，无构建工具、无服务器。目标是零门槛、开箱即玩的休闲消除体验。

## 目录结构

```
连连看/
├── 水果连连看.html   # 主游戏文件（最新版，以此为准）
├── index.html        # 备用入口（保留兼容）
├── PRD-连连看.md     # 产品需求文档（功能决策来源）
├── DESIGN.md         # 视觉设计规范（色板、间距、动效）
├── agents/           # 自定义 AI Agent 定义
└── slash/            # 自定义斜杠命令
```

## 技术约束

- 零依赖：不引入任何第三方库，不加构建步骤
- 单文件：所有 HTML/CSS/JS 写在同一个 `.html` 文件中
- 兼容性：目标支持 Chrome 90+、Safari 14+、Edge 90+

## 核心游戏规则（改动前必读）

- 连线折角 ≤ 2 次（经典规则，不可魔改）
- 棋盘边缘有一圈空格，供连线绕行
- 每次选中/消除/错误都需有明确视觉反馈（300ms 内响应）

## 常用操作

```bash
# 本地预览（推荐）
npx serve .

# 直接打开（快速验证）
open 水果连连看.html
```

## 设计参考

修改 UI 前先读 `DESIGN.md`，其中定义了完整色板（`--color-primary` 等 CSS 变量）、字体规范和动效时长，所有视觉改动必须与其保持一致。
